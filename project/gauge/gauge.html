Un petit projet comme ça pour s'amuser. Il s'agissait de réaliser une jauge verticale sans l'aide de CANVAS.
Les différentes sections montrent le cheminement parcouru depuis le début le plus basique (en bas) jusqu'à la fin la plus raffinée (avec du javascript pour faire le boulot)

Ce qui donne point de vue code HTML : 

    <!--Les DL m'ont paru ce qu'il y a de plus sémantique pour présenter une liste de clé/valeur
        Les classes ne servent pas au javascript et peuvent être changées -->
    <div class="gaugeContainer" id="jsGauge">
        <dl>
            <dt>Pouic</dt>
            <dd class="gaugePercent">100%</dd>

code JS : 

$(function(){
    // initialisation de la jauge
	$("#jsGauge dl").gauge({
		height: "7em", // Hauteur des barres
		background : "#FC0C8A", // Couleur de fond
		easing : "easeOutBounce"  // easing d'animation
	});
	// Méthode pour mettre à jour une jauge
	$("#jsGauge dl").gauge("setPercent",5);
	// Méthode pour retrouver la valeur d'une jauge
	$("#jsGauge dl").gauge("getPercent",5);
});
