  1. Suppression du framework elastic CSS
    27 12 2010

    Le framework elasticss permet grâce à l'ajout d'un fichier javascript de pouvoir facilement faire du multi colonnage. Le fait d'avoir un JS qui gère la présentation permet un rendu uniforme sur les différents navigateurs (ou presque).Je n'ai constaté que de léger soucis sous ie6 mais rien de bien grave.
    Le désavantage du framework est justement de ne pas reposer uniquement sur du css. Un problème dans un des codes JS et la présentation peut être complètement changée... Du coup j'ai décidé de me reposer sur :

    1. le CSS3 dont :
    2. les propriétés grid-rows, columns & grid-columns
    3. les media-queries
    4. un mode dégradé pour Internet Explorer.
